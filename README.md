# Stellar MCP Server

## Description

A simple MCP (Multi-Channel Protocol) server for interacting with the Stellar network.

## Prerequisites

*   Node.js
*   npm

## Installation

1.  Clone the repo.
2.  `npm install`

## Usage

1.  Configure: Copy `.env.example` to `.env` and adjust settings.
2.  `npm run build`
3.  `npm start`
